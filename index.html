AWS IoT MQTT Ball Tracker

This application visualizes ball position data from an AWS IoT MQTT topic.

Setup Instructions:

  1. Upload all 3 required certificate files:
    • Device certificate (.pem.crt)
    • Private key (.private.key)
    • Root CA certificate (AmazonRootCA1.pem)
  2. Enter the MQTT topic that publishes ball position data
  3. Select the field type that matches your sport
  4. Click "Connect to AWS IoT" to start receiving position data

Expected message format:

{
  "T": 0,
  "X": 51,  // 0-102 range
  "Y": 32,  // 0-64 range
  "P": 1,   // 1=home, 0=away
  "Pa": 0,  // Pass indicator
  "G": 0,   // Goal/Try indicator
  "C": 0    // Conversion
}
